# SepsisSense

AI-powered Early Sepsis Prediction using Machine Learning and Flask.

## 🌐 Live Demo
https://sepsissense.onrender.com

## 🚀 Features
- Early sepsis prediction using Machine Learning
- Logistic Regression model
- Flask backend
- Modern responsive web interface
- Healthy & Septic demo patients
