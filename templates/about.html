About SepsisSense

SepsisSense is an AI-powered clinical decision support application that predicts the risk of sepsis using machine learning based on 20 clinical features including vital signs and laboratory parameters.

Project Highlights

Machine Learning

Logistic Regression trained on the PhysioNet Sepsis Dataset.

Clinical Decision Support

Provides early risk prediction to support clinicians.

Technology

Python • Flask • Scikit-learn • HTML • CSS • JavaScript

⚠ Medical Disclaimer

This application is intended only for educational and clinical decision-support purposes. It should never replace professional medical judgement or diagnosis.
